## February 2026 - Auth Blocks

All [blocks](/blocks) now include a complete authentication suite.

- **7 auth blocks** - Login, Signup, Forgot Password, Reset Password, Verify Email, Two-Factor Auth, and OAuth Buttons.
- **3 variants each** - Every block ships with minimal, standard, and feature-rich variants.
- **CLI not yet available** - The CLI is still in development. For now, copy blocks directly from the code panel on each block page.

<PackageInstall command="npx @chadcn/cli add auth/login --variant standard" />

Browse the full collection on the [blocks](/blocks) page.

## February 2026 - CLI Coming Soon

The CLI is currently in development. For now, you can browse and copy blocks directly from the website.

- **Manual install** - Copy code from the block viewer's code panel into your project.
- **CLI planned for Phase 17** - The `add` and `init` commands will be fully functional soon.

Once ready, the workflow will be:

<PackageInstall command="npx @chadcn/cli add auth/login --variant standard" />
